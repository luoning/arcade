# 华容道赛博三国版 - 关卡设计与通关指南

兄弟，这里是华容道所有内置关卡的阵型设计和背景说明。
每一关的布局、难度及理论上的最少步数都记录在案，供你挑战时参考。

---

## 棋盘网格代号

棋盘为 4 列 x 5 行 的网格，坐标格式为 `[x, y]`。其中：
- `x` 的范围是 `0` 到 `3`（自左向右）
- `y` 的范围是 `0` 到 `4`（自上向下）

---

## 关卡详解

### 1. 横刀立马 (经典名阵)
- **战局难度**：⭐⭐⭐⭐⭐ (极难)
- **最少步数**：81 步
- **阵型示意**：
  ```
  [张飞] [曹操] [曹操] [赵云]
  [张飞] [曹操] [曹操] [赵云]
  [马超] [关羽] [关羽] [黄忠]
  [马超] [ 卒 ] [ 卒 ] [黄忠]
  [ 卒 ] [ 空 ] [ 空 ] [ 卒 ]
  ```
- **关卡说明**：华容道最著名的排局，也是千百年来流传最广的经典阵型。曹操被赵云、张飞、关羽等层层包围，脱逃路线极其漫长复杂，需要精细腾挪。

### 2. 指挥若定
- **战局难度**：⭐⭐⭐⭐ (较难)
- **最少步数**：65 步
- **阵型示意**：
  ```
  [张飞] [曹操] [曹操] [赵云]
  [张飞] [曹操] [曹操] [赵云]
  [ 空 ] [关羽] [关羽] [ 空 ]
  [马超] [ 卒 ] [ 卒 ] [黄忠]
  [马超] [ 卒 ] [ 卒 ] [黄忠]
  ```
- **关卡说明**：相比横刀立马，两个空位初始位于中间两侧。看似空位大，但极易造成局部死锁，要求玩家每一步都运筹帷幄。

### 3. 将拥蜀道
- **战局难度**：⭐⭐⭐⭐ (较难)
- **最少步数**：50 步
- **阵型示意**：
  ```
  [曹操] [曹操] [关羽] [关羽]
  [曹操] [曹操] [张飞] [赵云]
  [马超] [黄忠] [张飞] [赵云]
  [马超] [黄忠] [ 空 ] [ 空 ]
  [ 卒 ] [ 卒 ] [ 卒 ] [ 卒 ]
  ```
- **关卡说明**：大将挡在出口前方的蜀道上，关羽和曹操在最顶端对峙，需要利用底部的空挡先把小兵挪开，再将诸将逐一撤离。

### 4. 左右逢源
- **战局难度**：⭐⭐⭐ (普通)
- **最少步数**：39 步
- **阵型示意**：
  ```
  [张飞] [曹操] [曹操] [赵云]
  [张飞] [曹操] [曹操] [赵云]
  [ 卒 ] [ 空 ] [ 空 ] [ 卒 ]
  [马超] [关羽] [关羽] [黄忠]
  [马超] [ 卒 ] [ 卒 ] [黄忠]
  ```
- **关卡说明**：空位集中在棋盘中央，曹操左右都有退路，但小卒挡在关键节点，需要先拉空两翼，方可通关。

### 5. 兵临城下
- **战局难度**：⭐⭐ (简单)
- **最少步数**：25 步
- **阵型示意**：
  ```
  [ 卒 ] [张飞] [赵云] [ 卒 ]
  [ 卒 ] [张飞] [赵云] [ 卒 ]
  [马超] [曹操] [曹操] [黄忠]
  [马超] [曹操] [曹操] [黄忠]
  [ 空 ] [关羽] [关羽] [ 空 ]
  ```
- **关卡说明**：曹操已经突围到了第三排，四个小卒被合围在上方。曹操只需要避开关羽的防线就能破局，适合用来熟悉拖拽手感。
