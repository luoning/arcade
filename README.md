# 🎮 Neon Arcade // 赛博霓虹街机电玩城

欢迎来到 **Neon Arcade**！这是一个视觉极其炸裂、充满复古 Synthwave 律动与 CRT 扫描线质感的网页端街机游戏厅。

整个大厅由 **Vanilla JS (纯原生 JavaScript)**、HTML5 Canvas 与 CSS3 驱动，不依赖任何庞大的现代框架，轻量、纯粹且开箱即用。

---

## ✨ 项目特色 // FEATURES

1. **赛博朋克街机美学**：
   - 3D 悬浮质感的街机机台交互，带霓虹呼吸发光特效。
   - 仿真 CRT 电视屏幕滤镜，带横向扫描阴影，还原 80 年代街机机房观感。
   - 漂浮的霓虹微粒与背景 Synthwave 动态流光网格地板。
2. **纯代码合成 8-bit 音效**：
   - 摒弃了沉重的音频文件，由 **Web Audio API** 纯代码实时波形合成。
   - 还原 NES 经典的方波、三角波与噪音颤音，提供清脆的投币声、碰撞音及胜利得瑟乐段。
3. **统一大厅成就系统**：
   - 跨 IFrame postMessage 通信机制，游戏内达成特定条件（如扫雷通关、乒乓零封、数独熔毁等）实时向大厅推送并解锁成就。
   - 内置成就墙看板，支持本地 `localStorage` 存档记录。
4. **极致优化的操控交互**：
   - 自适应高分屏的高清 DPR (devicePixelRatio) 物理像素渲染，杜绝画面模糊。
   - 完美适配移动端触屏滑动手势，为移动玩家配备了防误触的专属虚拟“⚡ 蓄力/射击”控制按键。

## 🛠️ 怎么玩 // HOW TO RUN

1. 本项目为**纯前端静态项目**，无需搭建复杂的后端服务器。
2. 鼠标双击直接打开 `index.html` 即可在任意现代浏览器（Chrome / Edge / Firefox / Safari）中运行。
3. **推荐运行方式**：在 VS Code 中使用 `Live Server` 插件一键启动，以便在手机和电脑端同网段联机游玩。

## 🕹️ 核心重构版游戏介绍 // FEATURED DETAILS

### 🏓 等离子乒乓 (*Cyber Pong*)
- **垂直对攻**：1P 控制底部极板，2P/AI 控制顶部极板。球体在左右墙壁反弹。
- **物理摩擦**：极板在高速移动时擦球，会将极板的水平速度转移给球，形成曲线球（香蕉球）轨道。
- **蓄力与方向瞄准**：击球时按住空格（或鼠标/触屏长按），球会吸附在极板上进行充能。蓄力时射出**红外线指示器，且在墙壁上可以最多折射反射 3 段**。使用 `A / D` 或鼠标滑动可调整反射角度，释放即可打出金光特效的超级扣杀。

### ⛵ 大航海：无尽航路 (*Uncharted Voyage*)
- **港口贸易**：在 6 个特色港口之间跑商，买低卖高赚取差价。
- **改修升级**：在造船厂升级货舱容量、加固龙骨护甲、加装飞剪帆以减少风阻、加装大炮武装。
- **航海日志**：根据实时风向罗盘修正航道，海上概率遭遇雷暴、漂流物资以及突袭的海盗。

### 🎱 霓虹斯诺克台球 (*Neon Snooker*)
- **正式规则**：斯诺克对战模式提供标准的 15 颗红球、6 颗彩球摆法，严格执行红彩交替击打、彩球置点复位、犯规与罚分换人、手中的球 D 区击打等正式比赛判定。支持人机 VS AI 对局或本地双人同屏博弈。
- **特色模式**：激光雷区模式需让球体完美躲开桌面上的发光红色水雷；限时狂飙模式则提供 60 秒时间，击球进袋获得秒数奖励，挑战连击与积分上限。
- **Web Audio リアル音效**：通过代码合成真实球杆击球声、物理反弹碰球声和落袋坠落声。
