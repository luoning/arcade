# 🎨 战报战况与军区面板 UI 升级设计文档 (`ui_upgrade.md`)

本规范详细定义了伴随部众兵卒系统与军备重器下放至郡县后，前端展示层（侧边面板、战报日志、地图提示）的升级方案。

---

## 一、 郡县悬浮Tooltip升级

当玩家鼠标悬停在地图上的任意郡县时，展示的信息除原有的繁荣度之外，必须显示**基于三维矩阵计算出的历史融合称谓及部众数**：

```
+------------------------------------------+
|  【新野】 (刘备)                         |
|  --------------------------------------  |
|  繁荣度: 52,000 / 100,000                |
|  郡县驻军:                               |
|   - 役卒: 500 名                         |

|   - 大黄弩士: 50 卒   - 轻装骁骑: 10 骑   |

|  --------------------------------------  |
|  状态: 🕯️ 饱暖安宁                        |
+------------------------------------------+
```

## 二、 右侧军区微操调度面板重塑 (`rightPanel`)

点选特定郡县后，右侧操作面板不再显示抽象的“招募三兵种”现代按钮，全面改版为**“军事政令”**界面：

### 1. 简拔役卒与专业募兵
*   **【征发役卒】**：消耗少许资金与屯粮，直接扩充本地役卒数量。
*   **【简拔专业】**：点选役卒，消耗金钱将其编练简拔为“刀盾手”、“持矛兵”、“大戟士”、“弩兵”或“突骑兵”。

### 2. 升擢擢升与整备制式装备
*   **【百炼成钢】**：花费金贯，擢升本地某兵科部众等级（新卒 $\rightarrow$ 悍卒 $\rightarrow$ 精兵）。
*   **【披坚执锐】**：当兵科达到精兵时激活。花费金钱与矿石，为其编练【两当扎甲】、【百炼精铤】或【大黄弩】。

### 3. 督造重器与调防
*   **【筑墙御防】**：花费资金与役卒，建造【筌蹄】（阻遏骑兵）或【木兽】（焚烧步卒）。
*   **【督造攻器】**：花费资金，在工坊中监造【霹雳车】或【冲车】。
*   **【调遣移防】**：点选接壤的己方城池，滑动条划定兵卒数量，将其调派移防。

## 三、 古风历史感战报日志生成器 (`addLog`)

移除所有的现代游戏日志，全部战报由古风词汇组合生成：

### 战报示例一（远程压制）：
`[建安十三年春]` `曹操` 督率三军急袭 `刘备` 治下之【新野】。攻方【元戎弩手】万弩齐发，矢如雨下，压制了新野城头。刘备守军拼死坚守，曹操攻城失利退兵。

### 战报示例二（阵地突袭）：
`吕布` 麾下【并州骁骑】突袭【小沛】。敌军突骑来袭，小沛本地守军【持矛戟士】列拒马枪阵，枪刺如林，将飞将阻截于关外！

### 战报示例三（天子慕义）：
【许昌】有新卒听闻 `汉献帝` 下发勤王诏书，怀“思汉慕义”之心，连夜弃营潜逃，奔赴天子麾下。
